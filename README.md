# Demo SaaS de Automatizacion con IA

Demo visual de una plataforma SaaS para centralizar conversaciones, automatizar respuestas con IA, entrenar un bot y revisar metricas operativas.

Esta version es una demo inicial creada para comunicar la idea del producto. No incluye backend, autenticacion, base de datos, pagos ni integraciones reales. Todo funciona con datos simulados en el navegador.

## Modulos principales

- Dashboard: resumen visual de conversaciones, rendimiento del bot y metricas clave.
- Inbox: bandeja multicanal con chat y ficha del cliente.
- Automatizaciones: reglas visuales para responder, clasificar y escalar casos.
- Entrenamiento del Bot: configuracion conceptual de tono, conocimiento y documentos.
- Clientes: mini CRM con estado, interes, valor estimado y responsable.
- Canales: estado simulado de WhatsApp, Instagram, Web Chat, Messenger y Email.
- Analiticas: graficos de volumen, satisfaccion, resolucion y ventas asistidas.
- Simulador IA: laboratorio para probar intencion, sentimiento y respuesta sugerida.

## Ejecutar localmente

```bash
npm run dev
```

Luego abrir:

```text
http://localhost:5173
```

## Tecnologia

- HTML, CSS y JavaScript modular
- Datos mock locales
- Sin llamadas HTTP ni servicios externos
